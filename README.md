# Marks Calculator using Dunder Methods

## Project Overview

Marks Calculator using Dunder Methods is a Python project developed to understand how Python's special methods (Dunder Methods) work internally.

The project demonstrates how custom objects can interact with built-in Python operators and functions by implementing:

- `__init__()`
- `__str__()`
- `__repr__()`
- `__add__()`

This project was built as part of my Python OOP learning journey.
